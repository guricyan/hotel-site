# 网站内容修改说明

## 1. 修改文字

打开：

`src/data/siteContent.js`

这个文件集中保存了：

- 品牌名与首页标语
- 邮箱、微信
- 首页全部文案
- 五套房源的简介、人数、卧室、设备
- 附近景点与路线说明
- 所有图片路径

修改双引号内的文字即可，尽量不要删除逗号、方括号和花括号。

## 2. 替换照片

图片统一放在：

`public/images/`

首页照片：

`public/images/home/`

房源照片：

- `public/images/rooms/yousui/`
- `public/images/rooms/danxia/`
- `public/images/rooms/yotsuru/`
- `public/images/rooms/xianyun/`
- `public/images/rooms/tianyama/`

建议先将照片压缩成 JPG 或 WebP，单张控制在 300KB–800KB。

例如上传：

`public/images/rooms/yousui/living-room.webp`

然后在 `src/data/siteContent.js` 中把原路径：

`/images/rooms/yousui/01.svg`

改成：

`/images/rooms/yousui/living-room.webp`

## 3. 修改联系方式

在 `siteConfig.contact` 中替换：

```js
email: "你的邮箱",
wechat: "你的微信号",
```

邮箱替换后，“发送预订咨询”按钮会自动打开客人的邮件软件，并生成入住咨询模板。

## 4. 发布更新

在 GitHub 仓库中修改或上传文件，然后点击 Commit changes。

Cloudflare Pages 会自动开始新一轮部署，通常几分钟内更新。

## 5. 注意事项

- 景点距离与车程目前按照提供的文案整理，正式发布前请再次核对。
- 准确门牌不建议直接公开，应在预订确认后发送。
- 当前站点使用本地占位 SVG 图，替换成真实照片后视觉效果才会完整。
